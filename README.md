### Installation steps:

1. Clone the repo into local -  
```bash
git clone https://github.com/vineetj5/UMD_Event_DialogSystem.git
```
2. Change to the directory  to the project folder - 
```bash
cd UMD_Event_DialogSystem
```
3. Setup the OPENROUTER_API_KEY key in env file - 
```bash
OPENROUTER_API_KEY=your_key
```
4. Build the Docker Image and start the application - 
```bash
docker compose up --build
```
5. To open the app, click on this link
```bash
http://localhost:8501
```
6. To stop the container - 
```bash
docker compose down
```
